# Discord Automation Research Project by HIT

Este repositorio se publica **exclusivamente con fines educativos y de investigación**. El código presente forma parte de un **análisis técnico** sobre automatización en plataformas de mensajería y **no está destinado a su ejecución en entornos reales** ni a su uso contra servicios de terceros.

---

## Estado del proyecto

* **Proyecto archivado**
* **No mantenido**
* **No recomendado para uso práctico**
* **Sin soporte**

El contenido se conserva como referencia histórica y para el estudio de patrones de programación, manejo de eventos y estructuras de proyectos.

---

## Alcance y limitaciones

* No se proporciona documentación de uso, instalación o configuración.
* No se incluyen guías para interactuar con la API de Discord.
* No se aceptan contribuciones orientadas a funcionalidad operativa.
* Cualquier intento de uso real **no está respaldado** por el autor.

---

## Cumplimiento y responsabilidad

El autor **no fomenta ni aprueba** actividades que infrinjan los Términos de Servicio de Discord o cualquier otra plataforma.
El repositorio **no debe utilizarse** para automatización de cuentas de usuario, spam, raids, ni comportamientos abusivos.

El usuario es el único responsable del uso que haga del contenido.

---

## Uso permitido

* Lectura y análisis del código con fines educativos
* Estudio de arquitectura y estilo
* Aprendizaje teórico sobre eventos y control de flujo

---

## Uso no permitido

* Ejecución en cuentas reales
* Automatización de usuarios
* Interacción con servidores de terceros
* Cualquier forma de abuso o acoso
